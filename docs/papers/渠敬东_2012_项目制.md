# 项目制：一种新型国家治理体制

## 核心概念

@渠敬东 提出，**项目制**是改革开放以来形成的一种新型国家治理体制。

### 项目制的三层含义

1. **财政体制层面**: 专项资金的管理方式
2. **行政体制层面**: 跨部门、跨层级的协调机制
3. **社会治理层面**: 国家与社会互动的新形式

### 项目制的运行机制

```
中央专项转移支付 → 地方竞争申报 → 项目包装与执行 → 验收与绩效评估
```

## 项目制的社会效果

### 积极方面
- 突破科层制的条块分割
- 激发地方发展动力
- 提高资源配置效率

### 问题与挑战
- 基层配套资金压力
- 形式化的项目包装
- 短期行为与长期发展的矛盾
