# 基层政府间的共谋现象

## 核心观点

@周雪光 在这篇经典论文中，探讨了中国政府内部上下级政府间在应对上级政策指令时采取**共谋行为**的制度逻辑。

### 什么是共谋现象？

- **定义**: 上下级基层政府通过各种形式来应付上级政府的要求和检查
- **表现形式**: 数据造假、形式主义、选择性执行

### 共谋的制度根源

1. **委托-代理问题**
   - 信息不对称
   - 多层级委托-代理链条

2. **激励机制矛盾**
   - 压力型体制下的考核压力
   - 资源与目标的错配

3. **制度性空间**
   - 政策执行的模糊性
   - 非正式制度的运行
