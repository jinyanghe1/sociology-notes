# 半个世纪的农村制度变迁

## 核心论点

本文以"成本-收益"分析框架审视新中国50年农村制度变迁，提出**"小农村社经济"**这一核心概念，认为在"人地关系高度紧张"和"城乡二元结构"两大基本矛盾的制约下，中国农村形成了独特的土地村社所有制与农户承包经营权"两权分离"的制度安排。作者首次明确提出**"皇权不下县"**这一著名论断，主张通过**重建基层合作经济组织**和**税费改革**降低制度成本，缓解农民负担过重问题。

## 研究背景与问题

### 时代背景（1999年）

| 矛盾现象 | 具体表现 |
|---------|---------|
| **农业产量翻番** | 粮食从1949年410亿斤增至1万亿斤，人均占有量达世界平均水平 |
| **农民收入负增长** | 乡镇企业吸纳能力下降，农民外出打工困难，高利贷死灰复燃 |
| **制度成本过高** | 基层政府税费征收困难，乡村组织大量负债，农民负担沉重 |
| **市场化困境** | 户均0.41公顷的小农经济无法与WTO框架下的国际大农场竞争 |

### 核心问题

> "假如外部条件难以改变，我们能否走出这样'二、三十年再来一次'的恶性循环？"

土改（1949-1952）→ 大包干（1978-1982）→ 延包30年（1997-1999），每次土地按人均分间隔仅20-30年，土地日益"福利化"，农户经营规模细碎化，农业现代化目标与小农经济现实形成根本矛盾。

## 理论框架：两个基本矛盾

### 1. 人地关系高度紧张（内生性制约）

```
人口增长：4亿 → 9亿
人均耕地：减少一半以上
土地功能：生产资料 ↓↓↓ 社会保障功能 ↑↑↑
分配原则：效率 → 公平（按人均分）
制度结果：土地全面"福利化"
```

**历史连续性**：解放前即已形成"两田制"（村社公田+农民私田）与"两权分离"（地主占有权+农民租佃权），这种制度安排是**内生的、长期性**的，而非外来移植。

### 2. 城乡二元结构（体制性矛盾）

```
国家工业化 ←→ 小农经济剩余量少且分散
    ↓
政府与分散农民交易成本过高
    ↓
建立高度集体化制度作为统购统销的组织载体
    ↓
形成城乡分割对立的二元结构
```

## 制度变迁的历史逻辑

### 因果链条

```
人均资源有限的农业大国
    ↓
战争环境 + 无外部市场 → 内向型自我积累
    ↓
重工业优先的国家工业化
    ↓
从农业提取资本原始积累的需求
    ↓
与土改后小农经济矛盾 → 发动集体化
    ↓
重结构工业"资本增密、排斥劳动"
    ↓
城市工业无法充分吸纳就业 → 二元结构固化
    ↓
农业人口滞留农村 + 资本流入城市工业
    ↓
集体化内部人均剩余减少 → 以生存保障为前提的成员权分配原则
    ↓
劳动积极性下降 → 集体化难以为继
    ↓
大包干：再次按社区人口均分土地
    ↓
农户经营规模细碎化 + 兼业化
    ↓
【改革方向】加快城市化 + 小农村社内部制度建设
```

## "皇权不下县"：核心论断与学术对话

### 概念提出

> "由于小农经济剩余太少，自秦置郡县以来，历史上从来是'皇权不下县'。解放前县以下虽然设立派驻性质的区、乡公所，但并不设财政，不是一级完全政府。农村仍然维持乡村自治，地主和自耕农纳税，贫雇农则只交租。这种政治制度得以延续几千年的原因在于统治层次简单、冗员少，运行成本低。"

### 学术源流与对话

| 学者 | 核心概念 | 与本文的关系 |
|------|---------|-------------|
| @费孝通 | "双轨政治"（皇权+绅权并行） | 温铁军将现象凝练为"皇权不下县"，但**功能主义归因**（成本-效率）与费孝通的**文化-结构分析**有本质区别 |
| @梁漱溟 | "伦理本位，职业分途" | 士绅社会的自我调解机制 |
| @张静 | "双轨忠诚"（村干部的身份张力） | 质疑简单二元对立，强调国家与社会的交织 |
| @秦晖 | "大共同体本位" | **直接反驳**温铁军，认为传统中国并非"小共同体本位"，国家权力渗透远比"不下县"复杂 |

### 批判性反思

**理论张力**：
- 温铁军的解释是**功能性**的（降低治理成本）
- 但"自治"的可行性不仅取决于成本，更取决于**国家能力**和**社会内部公共性生产**
- 张静的研究表明，基层治理的关键并非"自治vs他治"的二元选择，而是**两种权威的交织与协商**

## 1990年代基层冗员的实证数据

### 规模数据（1999年前后）

| 指标 | 数据 | 来源/备注 |
|------|------|----------|
| **乡镇财政供养人员** | 约870-1600万人 | 学者估算，2003年国务院发展研究中心 |
| **乡镇行政编制（2002年）** | 115万人 | 中央机构编制委员会 |
| **乡镇事业编制（2002年）** | 841万人（含教师593万） | 中央机构编制委员会 |
| **超编情况** | 江西某贫困县：定编404人，实有882人（超编118%） | 实地调查，2005年 |
| **平均每个乡镇** | 230人以上吃"财政饭"和"事业饭" | 2002年统计 |

### "条条"系统膨胀

**"七所八站"结构**：
```
县级职能部门
    ↓ 垂直管理下伸
乡镇派出机构：财政所、税务所、工商所、派出所、司法所、土地所、农技站、兽医站...
    ↓
人事权在县里，工资由乡镇财政承担
    ↓
乡镇政府无法管理，但必须供养
```

### 制度成本的量化（以安徽太和县为例）

> 太和县是全国最早进行税费改革试点的县之一。

| 项目 | 金额 | 占比 |
|------|------|------|
| 农业税及附加（2000年） | 0.87亿元 | 100% |
| 财政供养人员工资支出 | 0.9955亿元 | 114% |
| **财政缺口** | **6000多万元** | **51.3%** |

**关键发现**：即使100%的农业税用于发工资，仍然不够！这还不包括办公经费、基建支出等。

### 温铁军的核心判断

> "过去地主收50%高额地租农民仍然抢着租地，现在规定5%的摊派提留农民都不满；可见当前反映很大的农民负担过重其实是现象，**现行制度的运行和维持费用过重才是实质问题**。"

## 研究设计与方法

### 方法论特点

| 特征 | 具体表现 | 评价 |
|------|---------|------|
| **宏观视角** | 将50年制度变迁置于工业化-现代化的宏观进程中考察 | 优势：把握结构性约束；局限：可能忽视微观行动者的能动性 |
| **成本-收益分析** | 以"制度费用"为核心解释变量 | 优势：简洁有力；局限：过度理性化，忽视政治逻辑 |
| **历史比较** | 对比解放前后、集体化前后、大包干前后的制度形态 | 优势：揭示制度连续性；局限：对制度断裂性关注不足 |
| **政策导向** | 明确提出"改革试验区"等政策建议 | 优势：实践 relevance；局限：政策可行性论证不足 |

### 经验基础

- **长期田野调查**：作者在1980年代即参与中央农村政策研究，主持多地"改革试验区"
- **历史档案**：对农村土地产权演变的历史梳理
- **政策参与**：1987年中央5号文件的起草背景

## 政策主张：农村基本经济制度建设

### 1. 产权制度改革

**"产权两级构造"**：
- 农民以最终所有者身份拥有委托权（一户一票选举）
- 村社经济组织作为代理人，拥有管理权和处置权

**制度创新形式**：
- 股田制、反租倒包、股份水、股份城
- 以土地为中心的社区股份合作制

### 2. 政治体制改革

**核心方案**：

```
乡级：改制为县财政直接开支的"乡公所"
    ↓
村级：全面贯彻《村民委员会自治法》
    ↓
经济组织：乡、村两级建立合作社作为独立经济法人和纳税主体
    ↓
征税方式：只对约4万个乡级、70万个村级合作社征收税费
    ↓
（不再对2亿多分散农户直接征收）
```

**预期效果**：
- 大幅降低交易成本（从对2亿农户征税→对74万合作社征税）
- 精简机构，分流人员至中心镇、合作社
- 基层管理成本下降后，试验"公粮制"改革

## 批判性评价

### 边际贡献

**理论贡献**：
1. **"小农村社经济"概念**：区别于土地私有制小农经济的本土化概念，强调村社所有制的历史连续性
2. **"皇权不下县"命题**：虽非首创，但在特定语境下将其工具化，成为解释农民负担的重要概念
3. **制度成本分析框架**：将新制度经济学的交易成本概念应用于中国农村研究

**经验贡献**：
1. 系统梳理1949-1999年农村制度变迁的宏观脉络
2. 提供1990年代末基层机构膨胀的详实数据
3. 为后续税费改革、新农村建设提供理论准备

### 逻辑严谨性评估

**优点**：
- 宏观视野开阔，能够把握"人地关系"这一根本约束
- 历史比较视角揭示了制度变迁的连续性
- 对"制度费用"的关注具有现实针对性

**局限与疑点**：

| 问题 | 具体表现 |
|------|---------|
| **过度决定论** | 将复杂制度变迁归因于"两个基本矛盾"，忽视政策选择的偶然性和政治博弈 |
| **理想化自治** | 假设小农是同质化的共同体，忽视村庄内部阶层分化（专业户、普通农户、贫困户） |
| **成本分析的局限** | 以"效率"为标准评判制度，但基层治理涉及合法性、公正性等不可通约的价值 |
| **政策可行性** | 重建合作社的方案假设了集体行动能力，但未充分论证在分化村庄中如何实现 |

### 社会价值与政策启示

**1999年的政策意义**：
- 为2000年启动的农村税费改革提供了理论依据
- "皇权不下县"概念影响深远，成为理解基层治理的经典框架
- 对"三农"问题的诊断具有前瞻性

**但历史发展超出了作者的预期**：

## 后税费时代的治理悖论（2006年后）

### 农业税取消后的意外后果

**温铁军的预期 vs 实际发生**：

| 温铁军（1999）的预期 | 实际发生（2006年后） |
|---------------------|---------------------|
| 取消税费 → 减轻负担 → 重建自治 | 取消税费 → 国家与农村关系"脱钩" → 自治功能萎缩 |
| 合作社成为纳税主体，承接治理功能 | 村庄"空心化"，劳动力外流，公共性难以重建 |
| 乡公所+村自治降低制度成本 | "无钱办事、无权办事"，基层"悬浮" |

### 吴重庆的观察：《孙村的路》

> "农业税取消后，村干部没有了与村民打交道的'由头'，也不再掌握任何可分配的资源。当需要推动基础设施建设时，面对村民的'搭便车'行为，几乎无能为力。"

**核心悖论**：
- 税费时代：村干部"一身两任"——既是国家代理人（收税），又是村庄当家人（分配资源）
- 后税费时代：国家资源输入，但**"权责不对称"**——责任层层下压，资源与权力留在上级

### @吴毅 的发现：《小镇喧嚣》

> "上面的政策是刚性的，但执行资源是弹性的。乡镇干部成了'螺蛳壳里做道场'，既要完成上级任务，又没有任何手段调动村民。"

### 项目制时代的治理困境（2010s-）

| 特征 | 表现 |
|------|------|
| **资源输入** | 国家通过项目制向农村转移支付 |
| **权责分离** | 上级部门"点菜"，基层政府"买单" |
| **形式主义** | 顶格管理、痕迹主义，基层忙于应付检查 |
| **去公共性** | 村民不再承担公共品供给责任，村庄政治萎缩 |

### 反思：温铁军方案的盲点

1. **假设了集体行动能力**：合作社方案忽视了**村庄分化**和**劳动力外流**的现实
2. **低估了国家权力渗透的复杂性**："皇权不下县"是一种理想型，但现代国家的技术治理（统计、监控、项目）已深入毛细血管
3. **忽视了治理的合法性维度**：治理不仅是成本问题，更涉及**谁之正义、何种权利**的规范性追问

## 与当代研究的关联

### 可借鉴之处

1. **宏观结构分析**："人地关系""城乡二元结构"仍是理解中国农村的基本框架
2. **历史视野**：制度变迁的路径依赖分析对理解当前土地制度改革仍有价值
3. **成本意识**：基层治理的财政约束仍是核心问题（尽管表现形式变化）

### 可改进/拓展的方向

1. **引入政治经济学视角**：不仅关注"成本-收益"，更关注**权力、利益集团、合法性**
2. **关注行动者能动性**：农民并非被动的制度接受者，而是有策略的行动者
3. **区域差异分析**："小农村社经济"在不同地区（东部vs西部、平原vs山区）有不同形态
4. **数字时代的治理**：移动互联网、大数据如何改变"国家-农民"互动模式？
