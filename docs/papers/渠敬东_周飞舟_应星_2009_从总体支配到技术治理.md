# 从总体支配到技术治理

## 核心论点

本文的核心贡献不在于提出"技术治理"这一概念本身，而在于揭示了一个**深刻的治理悖论**：

> 改革开放30年来，中国通过国家治理的"技术化"（行政理性化、程序规范化、专业科层化）试图走出"总体支配"的困境，但这种技术治理本身却内含了自我瓦解的逻辑——**程序技术与经营技术的结构性混同**。

这不是一个"转型成功"的叙事，而是一个**"没有替代方案"（TINA: There Is No Alternative）** 的无奈现实。当地方政府在财政压力下转向"经营城市"，当科层程序沦为寻租的遮羞布，技术治理展现出的不是韦伯式的理性铁笼，而是一个**精致的、计算化的、但本质上仍充满机会主义的治理形态**。

## 研究背景与问题

### 历史语境

文章发表于2009年，正值改革开放30周年。此时的中国面临双重挑战：

1. **内部挑战**：市场化改革带来的社会分化、治理碎片化、地方保护主义
2. **外部挑战**：全球金融危机后，国家干预经济的合法性需要新的论证

### 核心追问

作者团队提出的问题意识极具穿透力：

> 如果改革前的"总体支配"（政治动员、群众运动、意识形态控制）已被证明不可持续，那么**改革后的国家依靠什么来维系整合？**

答案是：**技术治理**——一种以财政资源为核心、以程序规范为手段、以效率为正当性基础的治理模式。

## 理论框架

### 从总体支配到技术治理：转型的三重机制

| 维度 | 总体支配（改革前） | 技术治理（改革后） |
|------|-------------------|-------------------|
| **权力基础** | 意识形态动员 | 行政程序合法性 |
| **资源配置** | 计划指令 | 财政包干/分税制 |
| **激励机制** | 政治忠诚 | 经济绩效（GDP竞赛） |
| **冲突调解** | 群众运动/阶级斗争 | 行政程序/法律诉讼 |
| **正当性来源** | 革命传统/领袖魅力 | 治理效率/发展绩效 |

### 财政改革的核心驱动

文章精准地识别了治理转型的**经济基础设施**：

**1. 财政包干制（1980-1993）**
- 核心逻辑："分灶吃饭"，地方成为相对独立的财政主体
- 治理效果：激活了地方发展动力，但也造成"诸侯经济"、市场分割
- 与 @周飞舟 后续研究对话：这一时期奠定了地方政府"经营企业"的行为模式

**2. 分税制（1994-至今）**
- 核心逻辑：中央上收财权、下放事权
- 治理效果：强化了中央宏观调控能力，但造成基层财政困难
- 意外后果：土地财政的兴起，地方政府从"经营企业"转向"经营城市"

### 技术治理的双重性

文章最具洞察力的概念辨析在于区分了两种"技术"：

**程序技术**（Procedural Techniques）
- 目标：行政过程的规范化、透明化、可预期
- 手段：目标责任制、考核指标体系、行政审批流程优化
- 理想类型：韦伯式科层理性

**经营技术**（Managerial Techniques）
- 目标：资源最大化汲取与配置效率
- 手段：项目包装、土地开发、招商引资
- 逻辑：新公共管理的企业家政府

### 核心悖论：程序技术与经营技术的混同

文章最尖锐的批判出现在结尾部分（这也是 @用户 指出的关键段落）：

> "事实上，在某些基层行政中虽天天制度上墙、数字上表，却依然盛行着'潜规则'，成为大量寻租得以发生的温床...政府行为虽在形式上发生了由行政经营型向行政服务型的转化，但在具体项目的申报和实施过程中，仍按照经营化和产业化的标准和逻辑来推展。"

这种**混同**并非偶然失序，而是结构性的：

1. **财政压力下的路径依赖**：分税制后基层政府财权事权不匹配，"经营"成为生存必需
2. **考核体系的扭曲**：GDP导向的绩效评估将程序理性转化为"数字出官"的投机
3. **程序的空心化**：制度上墙、台账完备，但实质决策仍遵循非正式政治

## 批判性评价

### 边际贡献

**理论贡献：**
1. **概念创新**：提出"技术治理"作为理解中国改革后国家治理的核心范畴，与 @福柯 的治理术（governmentality）、@韦伯 的理性化形成对话
2. **机制阐明**：清晰展示了财政改革如何作为"无声的革命"重塑国家-社会关系
3. **悖论揭示**：指出技术治理并非线性进步，而是内含自我解构的张力

**经验贡献：**
1. 提供了改革开放30年治理转型的宏观历史叙事
2. 识别了项目制、目标责任制等具体治理技术的运作逻辑
3. 为后续大量实证研究（如 @渠敬东 2012年项目制研究、@周雪光 2017年《制度逻辑》）奠定了问题意识

### 逻辑严谨性评估

**优点：**
- 历史制度主义视角，兼顾结构约束与行动者能动性
- 财政社会学视角，从"钱从哪里来、到哪里去"理解治理模式
- 概念辨析清晰，程序技术/经营技术的区分具有分析力

**局限/疑点：**
1. **因果关系过于简化**：财政改革是治理转型的"充分条件"还是"触发因素"？是否忽略了官僚体系自我演化、社会抗争倒逼等其他机制？
2. **二元框架的僵化**："总体支配"与"技术治理"的边界是否如此清晰？@周雪光 后来提出的"运动式治理"恰恰显示了二者的混合性
3. **缺乏行动者视角**：侧重于国家视角，对民众如何"被治理"、如何反抗/利用治理技术的分析不足（@应星 后续的上访研究部分弥补了这一缺陷）

### 2026年回望：哪些问题尘埃落定，哪些追问仍然回响？

**已尘埃落定的：**
- 技术治理作为国家治理核心特征的判断得到验证
- 财政改革驱动治理转型的机制被后续研究反复确认
- "程序-经营混同"的现象在土地财政、PPP项目、专项债等领域持续存在

**仍在回响的追问：**

**追问1：TINA困境**
文章暗示技术治理存在内在缺陷，但未能回答：**如果不走技术治理的路，还有什么替代方案？**

2026年的现实是：技术治理不仅没有衰退，反而通过数字化（健康码、社会信用体系、预算绩效管理一体化平台）得到了强化。这不是因为技术治理"好"，而是因为**没有替代性的制度方案能够整合如此大规模、高复杂度的社会**。

**追问2：基层政府的"技术性破产"**
文章预感到了基层财政困难，但未能预见其极端形式：当地方政府债务率超过警戒线、连偿债都力不从心时，技术治理的根基何在？

2020年代的"财政紧平衡"使这一问题愈发尖锐：
- 技术性破产的地方政府如何维持合法性？
- "有所为有所不为"的改革能否真正落实，还是只是财政压力下的被动收缩？

**追问3：程序理性与实质正义的张力**
文章提到的"制度上墙、数字上表、潜规则盛行"在2020年代演变为更精致的形态：
- 形式完备的预算绩效管理与实质上的"钱等项目"
- 层层把关的审批程序与实质上的"领导批示"
- 全覆盖的监督检查与实质上的"选择性执行"

**这是官僚机构的原罪，还是可改进的阵痛？** 文章倾向于前者，但2026年的政策实践仍在尝试后者（零基预算改革、财政治理数字化）。

## 与后续研究的对话

### 与 @渠敬东 后续研究的对话
2012年《项目制：一种新的国家治理体制》可视为本文的延续与修正：
- 项目制正是技术治理的典型形态（程序规范、竞争分配、绩效考核）
- 但渠敬东更强调项目制的**社会后果**：专项化治理对社会团结的侵蚀

### 与 @周雪光 的对话
《中国国家治理的制度逻辑》（2017）提出了"控制权"理论，修正了本文的二元框架：
- 不是"总体支配→技术治理"的替代，而是**行政发包制下的多元控制机制并存**
- "运动式治理"的存在说明总体支配的逻辑从未退场，只是被技术治理包裹

### 与 @应星 的对话
应星后续的上访研究（《大河移民上访的故事》）从底层视角补充了本文：
- 技术治理对普通人而言意味着什么？
- 信访制度的悖论（越级上访的自我实现）暴露了技术治理的边界

### 与 @周飞舟 的对话
周飞舟的"土地财政"系列研究深化了本文的财政视角：
- 技术治理的经济基础从"经营企业"转向"经营城市"
- 这一转变如何重塑了中央-地方关系、政府-农民关系？

## 与自身研究的关联

### 可借鉴之处

1. **财政社会学的视角**：从"钱"的流动理解权力关系，这一分析路径极具穿透力
2. **历史制度主义的方法**：将特定治理技术置于改革的历史脉络中理解
3. **批判性建构的态度**：不是简单肯定或否定技术治理，而是揭示其内在张力

### 可改进/拓展的方向

1. **加入数字治理的维度**：2009年的作者无法预见AI和大数据如何重塑技术治理，今天需要在"数字利维坦"与"数字民主"的张力中重新审视
2. **基层财政危机的深化分析**：文章预感了财政困难，但2020年代的债务危机需要更系统的政治经济学分析
3. **比较视野**：技术治理是否是中国特有？与西方新公共管理运动、发展型国家的比较可以更清晰地定位中国经验的独特性
