# 文档模板

## 论文笔记模板

```markdown
---
title: "论文标题"
authors: ["作者1", "作者2"]
year: 2023
tags: ["标签1", "标签2", "标签3"]
venue: "期刊名"
category: "papers"
summary: "一句话概括论文的核心贡献"
---

# 论文标题

## 核心观点

简要概括论文的核心论点。

### 理论框架

- 理论1
- 理论2

### 主要发现

1. 发现1
2. 发现2

## 与相关研究的对话

本文与 @作者名 的 @相关概念 形成对话。

## 关键引用

> 重要引用内容

## 相关概念

- @概念1
- @概念2
```

## 概念笔记模板

```markdown
---
title: "概念名称"
authors: ["概念提出者"]
year: 2014
tags: ["标签1", "标签2"]
category: "concepts"
summary: "概念的核心定义"
---

# 概念名称

## 概念定义

**概念名称** 是指...

### 核心内涵

1. 要点1
2. 要点2

## 理论发展

- 起源
- 演变
- 应用

## 相关概念

- @相关概念1
- @相关概念2
```

## 关联语法说明

### @ 语法

使用 `@` 符号创建文章间关联：

- `@作者名` - 关联到作者的其他作品
- `@文章标题` - 关联到具体文章
- `@概念名` - 关联到概念解释

### 示例

```markdown
本文受 @韦伯 的官僚制理论启发，
与 @周雪光 的 @行政发包制 形成对话。
```
