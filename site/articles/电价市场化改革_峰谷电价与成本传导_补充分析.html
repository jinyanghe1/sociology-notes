峰谷电价与成本传导率

对《电价市场化改革与市场势力》的方法论补充与批判性分析
⚠️ 重要方法论问题

🎯 核心问题：论文是否忽略了峰谷电价的异质性？

用户的敏锐观察

宋枫等（2025）使用广东省电力市场小时级高频数据估计成本传导率为55.7%，但可能存在一个重要疏漏：没有分时段（峰/平/谷）估计成本传导率。

❗ 为什么这是一个严重问题？

电力市场的峰谷时段具有截然不同的特征：

  • 高峰时段：需求刚性大（弹性小）、供给紧张、市场势力强
  • 低谷时段：需求弹性大、供给充裕、竞争激烈
  • 成本传导机制完全不同：峰谷应该分别估计，而非简单平均
📊 论文的做法
  • 使用小时级数据
  • 估计整体成本传导率：55.7%
  • 分析异质性：竞争程度、需求弹性、边际成本弹性
  • 但未明确区分峰谷时段
💡 建议的改进
  • 按峰/平/谷时段分别估计
  • 分析时段间传导率差异
  • 考虑跨时段替代效应
  • 评估峰谷电价政策效果

📚 理论分析：为什么峰谷时段的成本传导会不同？

成本传导率的基本公式（寡头竞争模型）：

$$\\rho = \\frac{dP}{dMC} = \\frac{1}{1 + \\frac{1}{|\\epsilon_D| \\cdot \\mu}}$$

其中：$\\rho$ = 成本传导率，$\\epsilon_D$ = 需求价格弹性，$\\mu$ = 勒纳指数（市场势力指标）

🔍 关键洞察

根据上述公式，成本传导率取决于两个关键因素：

  1. 需求价格弹性 |ε_D|：弹性越小，传导率越低（企业有更大定价权）
  2. 市场势力 μ：市场势力越强，传导率偏离100%越多

峰谷时段这两个因素截然不同！

特征 高峰时段 (Peak) 低谷时段 (Off-peak)
需求弹性 小（刚性大，用户难以调整） 大（弹性大，可推迟用电）
供给状况 紧张（接近产能上限） 充裕（有大量闲置产能）
市场势力 强（少数机组决定价格） 弱（众多机组竞争）
成本传导率（理论预测） 较低（企业保留利润空间） 较高（接近完全竞争）
电价水平 高 低

📈 实证结果：分时段 vs 整体估计

📊 模拟结果解读

假设论文整体估计的成本传导率为55.7%，但如果分时段估计，可能会发现：

  • 高峰时段：传导率可能只有 30-40%（市场势力强，企业保留利润）
  • 平段：传导率约 50-60%
  • 低谷时段：传导率可能高达 70-90%（竞争激烈，接近完全传导）

结论：整体估计的55.7%掩盖了重要的时段异质性，低谷时段的竞争可能远比平均水平充分！

⚠️ 对论文结论的影响

1. 市场势力评估可能被低估

如果只报告平均55.7%，而高峰时段实际只有35%，说明在关键时刻（供给紧张）市场势力的危害被低估了。

2. 政策建议需要细化

"加强市场竞争"的建议应该具体化为：在高峰时段引入更多竞争机制（如需求响应、储能）。

3. 碳成本传导可能被误判

论文发现碳成本传导率42.9%不显著。但如果分时段看，可能高峰时段传导更充分（边际机组是煤电）。

4. 峰谷电价政策效果

分时段分析可以评估现有峰谷电价机制是否有效引导了需求，以及成本节约如何分配。

💡 对论文的改进建议

📝 方法论改进建议
  1. 分时段回归：将小时数据按峰/平/谷分类，分别估计成本传导率
    Price_t = α + β_peak · MC_t · D_peak + β_offpeak · MC_t · D_offpeak + controls
  2. 时段交互项：检验成本传导率在不同时段的差异是否显著
  3. 跨时段分析：考虑峰谷电价联动（如储能套利）对成本传导的影响
  4. 政策评估：评估峰谷电价机制是否有效降低了高峰时段的市场势力

建议的扩展模型（分时段成本传导）：

$$P_t = \\alpha + \\sum_{s \\in \\{peak, offpeak\\}} \\beta_s \\cdot MC_t \\cdot D_{s,t} + \\gamma X_t + \\epsilon_t$$

其中 $D_{s,t}$ 是时段虚拟变量，$\\beta_s$ 是分时段成本传导率

🎓 总结

核心结论

你的观察非常敏锐！宋枫等（2025）的论文确实存在一个潜在的方法论局限：没有充分考虑峰谷电价对成本传导率的影响。

电力市场的峰谷时段具有截然不同的市场结构：

  • 高峰时段：需求刚性、供给紧张、市场势力强 → 成本传导率应该更低
  • 低谷时段：需求弹性、供给充裕、竞争激烈 → 成本传导率应该更高

整体估计的55.7%可能掩盖了这些重要差异。分时段分析不仅可以验证论文结论的稳健性，还能为精准监管提供更丰富的政策启示。
