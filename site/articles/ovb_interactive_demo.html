🎮 遗漏变量偏误（OVB）交互式演示

基于张子尧、黄炜《实证研究中的控制变量选择：原理与原则》

📚 核心概念：什么是遗漏变量偏误？

场景：你想研究"教育对收入的影响"，但忽略了"能力"这个因素。

问题：能力强的人 → 更可能接受更多教育，同时 → 即使不受这么多教育也能挣更多钱。

后果：你把"能力带来的收入优势"误以为是"教育带来的收入优势"。

公式表达：

估计偏误 = \( c \times \lambda \)

其中：c = 遗漏变量对结果的效应，λ = 遗漏变量对处理变量的效应

🧮 逐步推导：偏误是如何产生的？

1 真实的数据生成过程（上帝视角）

收入 \( Y = \alpha + \tau D + cX + \epsilon \)

D = 教育年限，X = 能力，τ = 教育的真实效应（我们想要估计的），c = 能力的效应

2 研究者实际跑的回归（遗漏了能力X）

收入 \( Y = \alpha^* + \tau^* D + \text{error} \)

注意：τ* 是我们实际估计的系数，它可能不等于真实的 τ

3 OLS估计量的分解

当我们用OLS估计 τ* 时：

\( \hat{\tau}^* = \frac{\text{Cov}(Y, D)}{\text{Var}(D)} \)

4 代入真实的Y

把 \( Y = \alpha + \tau D + cX + \epsilon \) 代入上式：

\( \hat{\tau}^* = \frac{\tau\text{Var}(D) + c\text{Cov}(X,D)}{\text{Var}(D)} \)

5 最终结果：遗漏变量偏误公式

\( \hat{\tau}^* = \tau + \underbrace{c \times \frac{\text{Cov}(X,D)}{\text{Var}(D)}}_{\text{偏误项}} \)

或者简化为：估计值 = 真实值 + (能力效应 × 能力对教育的效应)

🎮 交互式参数控制

调整以下参数，观察遗漏变量偏误如何变化：

c (能力对收入的效应): 3.0

能力强的人收入高多少？（比如：能力每高1分，收入高3元）

λ (能力对教育的效应): 0.30

能力强的人多受多少教育？（比如：能力每高1分，多受0.3年教育）

⚠️ 关键洞察：偏误的方向和大小

  • 当 c 和 λ 同号时（都正或都负）：偏误为正 → 高估处理效应
  • 当 c 和 λ 异号时（一正一负）：偏误为负 → 低估处理效应
  • 当 λ = 0 时：能力不影响教育 → 无偏误（X不是混淆变量）
  • 当 c = 0 时：能力不影响收入 → 无偏误（X与结果无关）

🔗 与张子尧、黄炜论文的连接

本文演示的是论文中的核心观点：

"在因果识别环节，'好'控制变量通过合理分层让观测性数据在局部尽可能近似于随机化实验， 而'坏'控制变量会引入选择性偏误。"

这里的 X（能力）就是一个必须控制的"好"控制变量，因为它是：

  1. 处理前变量（在教育之前就已存在）
  2. 同时影响处理（教育）和结果（收入）的混淆变量

不控制它，就会产生我们上面演示的遗漏变量偏误。
