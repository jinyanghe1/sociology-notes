🎮 张子尧_黄炜_2025_实证研究中的控制变量选择：原理与原则

基于张子尧、黄炜（2025）《实证研究中的控制变量选择：原理与原则》

关键词：控制变量选择、遗漏变量偏误、因果识别、潜在结果框架、好控制变量、坏控制变量、DAG

本文保持“概念—推导—模拟—解释”的原有框架，通过交互式参数演示，帮助读者在直观观察中理解偏误如何产生、何时加剧、以及如何通过合理控制变量降低识别风险。

📚 核心概念：什么是遗漏变量偏误？

场景：我们希望识别“教育对收入的影响”，但模型中遗漏了“能力”这一关键因素。

问题：能力更强的个体往往更可能接受更多教育，同时即便教育年限相同，也可能获得更高收入。

后果：如果不显式处理能力因素，就会把“能力带来的收入差异”误判为“教育带来的因果效应”。

公式表达：

估计偏误 = \( c \times \lambda \)

其中：c = 遗漏变量对结果的效应，λ = 遗漏变量对处理变量的效应

🧮 逐步推导：偏误是如何产生的？

在明确直观机制后，下面通过标准 OLS 分解展示偏误项如何进入估计系数，并据此理解“为何必须控制混淆变量”。

1 真实的数据生成过程（上帝视角）

收入 \( Y = \alpha + \tau D + cX + \epsilon \)

D = 教育年限，X = 能力，τ = 教育的真实效应（我们想要估计的），c = 能力的效应

2 研究者实际跑的回归（遗漏了能力X）

收入 \( Y = \alpha^* + \tau^* D + \text{error} \)

注意：τ* 是我们实际估计的系数，它可能不等于真实的 τ

3 OLS估计量的分解

当我们用OLS估计 τ* 时：

\( \hat{\tau}^* = \frac{\text{Cov}(Y, D)}{\text{Var}(D)} \)

4 代入真实的Y

把 \( Y = \alpha + \tau D + cX + \epsilon \) 代入上式：

\( \hat{\tau}^* = \frac{\tau\text{Var}(D) + c\text{Cov}(X,D)}{\text{Var}(D)} \)

5 最终结果：遗漏变量偏误公式

\( \hat{\tau}^* = \tau + \underbrace{c \times \frac{\text{Cov}(X,D)}{\text{Var}(D)}}_{\text{偏误项}} \)

或者简化为：估计值 = 真实值 + (能力效应 × 能力对教育的效应)

⚠️ 关键洞察：偏误的方向和大小

结合上面的模拟结果，可以将偏误规律概括为以下四条判断规则：

  • 当 c 和 λ 同号时（都正或都负）：偏误为正 → 高估处理效应
  • 当 c 和 λ 异号时（一正一负）：偏误为负 → 低估处理效应
  • 当 λ = 0 时：能力不影响教育 → 无偏误（X不是混淆变量）
  • 当 c = 0 时：能力不影响收入 → 无偏误（X与结果无关）

🔗 与张子尧、黄炜论文的连接

回到论文语境，以上演示对应的是其关于“好控制变量”与“识别偏误”的核心论点：

"在因果识别环节，'好'控制变量通过合理分层让观测性数据在局部尽可能近似于随机化实验， 而'坏'控制变量会引入选择性偏误。"

这里的 X（能力）就是一个必须控制的"好"控制变量，因为它是：

  1. 处理前变量（在教育之前就已存在）
  2. 同时影响处理（教育）和结果（收入）的混淆变量

不控制它，就会产生我们上面演示的遗漏变量偏误。

🎯 第二部分：现代因果推断革命 —— 潜在结果框架

基于 Neyman-Rubin 潜在结果框架 (Potential Outcomes Framework)

🎭 核心思想：什么是"潜在结果"？

基本问题：一个人不可能同时"接受处理"又"不接受处理"。这就是因果推断的基本问题（The Fundamental Problem of Causal Inference）。

潜在结果（Potential Outcomes）的定义：

  • \(Y_i(1)\)：个体 \(i\) 接受处理时的潜在结果
  • \(Y_i(0)\)：个体 \(i\) 不接受处理时的潜在结果

关键洞察：对于每个个体，我们只能观察到其中一个结果（接受处理的观察到 \(Y_i(1)\)，没接受的观察到 \(Y_i(0)\)），另一个结果永远缺失。

个体处理效应 (ITE, Individual Treatment Effect)：

\( \tau_i = Y_i(1) - Y_i(0) \)

这是个体层面的因果效应，永远无法直接观测，因为 \(Y_i(1)\) 和 \(Y_i(0)\) 不可能同时存在。

📊 三类平均处理效应：ATE, ATT, ATU

既然个体效应无法观测，我们转向群体平均。根据 averaging 的人群不同，有三种主要 estimand：

名称 符号 公式 含义 可识别性
平均处理效应 ATE \( \tau_{ATE} = E[Y(1) - Y(0)] \) 对全体人群随机分配处理的期望效应 通常需要最强假设
处理组平均处理效应 ATT \( \tau_{ATT} = E[Y(1) - Y(0) | D=1] \) 对实际接受处理者的期望效应 政策评估中最常用
对照组平均处理效应 ATU \( \tau_{ATU} = E[Y(1) - Y(0) | D=0] \) 对实际未接受处理者的期望效应 常用于异质性分析

🔍 三者关系

\( \tau_{ATE} = \tau_{ATT} \times P(D=1) + \tau_{ATU} \times P(D=0) \)

ATE 是 ATT 和 ATU 的加权平均。当处理效应同质（所有人的 τ 相同）时，ATE = ATT = ATU。

⚠️ 简单差异均值（SDM）的问题

1 研究者实际能计算的是什么？

SDM (Simple Difference in Means)：

\( \hat{\tau}_{SDM} = \underbrace{E[Y|D=1]}_{\text{处理组观测均值}} - \underbrace{E[Y|D=0]}_{\text{对照组观测均值}} \)

2 展开处理组的观测均值

对于处理组（\(D=1\)），他们实际接受处理，所以我们观察到的是 \(Y(1)\)：

\( E[Y|D=1] = E[Y(1)|D=1] \)

3 展开对照组的观测均值

对于对照组（\(D=0\)），他们没有接受处理，但我们需要知道如果他们接受了会怎样（\(Y(1)\)）：

\( E[Y|D=0] = E[Y(0)|D=0] \)

但 SDM 用这个代替了 \(E[Y(0)|D=1]\)（处理组的反事实）！

4 SDM 的分解（核心公式）

\( \underbrace{E[Y|D=1] - E[Y|D=0]}_{\text{SDM 估计量}} = \underbrace{E[Y(1)-Y(0)|D=1]}_{\text{ATT（我们想要的）}} + \underbrace{\{E[Y(0)|D=1] - E[Y(0)|D=0]\}}_{\text{选择性偏误}} \)

选择性偏误 = 处理组和对照组在未接受处理时的潜在结果本来就不同。

🚨 直观理解选择性偏误

例子：研究"上北大是否能提高收入"

  • 上北大的人（处理组）：能力本身就强，即使不上北大也能挣不少钱
  • 没上北大的人（对照组）：能力分布不同，可能本来挣得就少
  • 选择性偏误：我们比较的其实是"能力强的人"vs"能力弱的人"，而非"上北大"vs"没上北大"的效果

公式表达： 选择性偏误 = \(E[Y(0)|D=1] - E[Y(0)|D=0]\)

如果处理组和对照组在基线（不接受处理时）就不同，SDM 就会产生偏误。

✅ 消除选择性偏误：三大关键假设

要正确识别因果效应（特别是ATE和ATT），需要以下假设：

假设 名称 数学表达 直观含义
SUTVA 一致性 + 无干扰 \(Y_i = Y_i(1)D_i + Y_i(0)(1-D_i)\) 处理是well-defined的；一个人的处理不影响其他人的结果
Ignorability 可忽略性/条件独立 \(\{Y(0), Y(1)\} \perp D | X\) 控制了X后，处理分配与潜在结果独立（无未观察到的混淆因素）
Overlap 正值性/重叠 \(0 < P(D=1|X) < 1\) 对于每种X，都有人接受处理也有人未接受（避免外推）

🎯 重点：可忽略性假设（Ignorability）

这是最关键的假设。它说：

给定观察到的协变量X，处理分配就像是随机的一样。

换句话说，所有影响处理和结果的混淆因素都包含在X中了。

数学推导：在可忽略性下

\( E[Y(1)|D=1, X] = E[Y(1)|X] \) （控制了X后，D与Y(1)独立）
\( E[Y(0)|D=0, X] = E[Y(0)|X] \) （同理）

因此，条件ATE可以识别：

\( \tau(x) = E[Y|D=1, X=x] - E[Y|D=0, X=x] = E[Y(1)-Y(0)|X=x] \)

然后对X求期望得到ATE：

\( \tau_{ATE} = E_X[\tau(X)] \)

⚠️ 重叠假设（Overlap）的重要性

即使满足可忽略性，如果没有重叠，我们仍然无法识别因果效应。

极端例子：研究"上北大对收入的影响"

  • 如果某些特征X（如高考满分）的人必然上北大（P(D=1|X)=1）
  • 对于这些X，我们观察不到"不上北大"的反事实 \(Y(0)\)
  • 必须外推其他X的人群的效应到这群人
  • 这种外推依赖模型假设，不可检验

实践建议：检查倾向得分（Propensity Score）分布，去掉没有共同支撑的区域。

🔗 连接第一部分：控制变量的作用

现在我们可以用潜在结果框架重新理解控制变量的作用：

控制变量的目的：让可忽略性假设成立。

具体来说：

  1. 不控制X：\(\{Y(0), Y(1)\} \not\perp D\)，处理组和对照组的 \(Y(0)\) 分布不同 → 有选择性偏误
  2. 控制X后：\(\{Y(0), Y(1)\} \perp D | X\)，在X的每个取值内，处理近似随机 → 无选择性偏误

这就是为什么张子尧、黄炜强调：控制变量的选择应该基于因果结构——我们需要控制所有影响处理和结果的混淆因素，使条件独立性成立。

📚 本章小结

  • 潜在结果框架：因果推断的基础，定义 \(Y(1)\) 和 \(Y(0)\) 为个体的两种潜在状态
  • 三类效应：ATE（总体）、ATT（处理组）、ATU（对照组），研究者需根据问题选择合适的目标
  • 选择性偏误：SDM 的问题在于比较了基线不同的群体，\(E[Y(0)|D=1] \neq E[Y(0)|D=0]\)
  • 识别假设：SUTVA（一致性+无干扰）、可忽略性（无未观察混淆）、重叠（正值性）
  • 控制变量的目的：通过控制混淆因素X，使可忽略性假设成立，从而消除选择性偏误

🚫 第三部分：坏控制变量 —— 当控制变量「反噬」你的估计

以「大学教育对收入的影响」为例，展示控制变量如何「好心办坏事」

⚠️ 核心警告：不是所有前定变量都应该被控制！

传统计量经济学教科书告诉我们：「控制越多变量越好」「控制前定变量总是安全的」。

这是错误的！

有些变量，即使它们在处理之前就已存在，控制它们反而会引入偏误。这些就是「坏控制变量」（Bad Controls）。

🎓 案例设定：大学教育对收入的影响

研究问题：上大学能否提高个人收入？

真实因果效应：\(\tau = 10\)（上大学使收入增加10个单位）

因果结构：

能力/家庭背景 ─→ 大学教育 ─→ 职业类型 ─→ 收入
                      ↓              ↑
                      └──────── 社交网络 ─┘

我们将展示三种「坏控制变量」如何扭曲估计：

  1. 中介变量：职业类型（大学→职业→收入）
  2. 对撞变量：社交网络（被教育和收入共同影响）
  3. M-偏倚变量：地理位置（复杂的混淆结构）

🔴 第一类坏控制：中介变量（Mediator）

1 什么是中介变量？

中介变量位于因果路径上，是处理影响结果的机制。

\( X \rightarrow \underbrace{M}_{\text{中介}} \rightarrow Y \)

2 案例：职业类型作为中介

大学教育 → 获得高薪职业（如金融、科技）→ 收入提高

职业类型就是中介变量！它解释了「为什么大学教育能提高收入」。

3 数学直觉：控制中介会发生什么？

假设真实模型：

\( M = \alpha_0 + \alpha_1 X + \epsilon_M \)（教育决定职业）
\( Y = \beta_0 + \beta_1 X + \beta_2 M + \epsilon_Y \)（教育和职业决定收入）

真实的总效应应该是：

\( \text{总效应} = \underbrace{\beta_1}_{\text{直接效应}} + \underbrace{\alpha_1 \beta_2}_{\text{间接效应（通过M）}} \)

但如果你控制了 M，你估计的只是直接效应 \( \beta_1 \)， 忽略了间接效应 \( \alpha_1 \beta_2 \)！

🎯 何时控制中介是「对」的？

如果你想知道总效应（上大学总共能提高多少收入），不要控制职业。

如果你想知道直接效应（除去职业渠道，上大学还能如何提高收入），可以控制职业。

关键：明确你的研究目标！大多数政策评估关心的是总效应。

🔴 第二类坏控制：对撞变量（Collider）

1 什么是对撞变量？

对撞变量是被X和Y共同影响的变量。

\( X \rightarrow \underbrace{C}_{\text{对撞}} \leftarrow Y \)

注意箭头的方向！与混淆变量（X←Z→Y）相反。

2 案例：社交网络作为对撞变量

想象一下「精英社交网络」：

  • 上大学 → 更容易进入精英社交圈
  • 收入高 → 更容易进入精英社交圈

社交网络就是对撞变量！它被教育和收入共同影响。

3 数学直觉：控制对撞为什么会引入偏误？

假设教育和收入本来没有关系（真实效应为0），但都影响社交网络：

\( C = \gamma_0 + \gamma_1 X + \gamma_2 Y + \epsilon_C \)

如果你控制C（只看C相同的人群），会发生什么？

如果某人C很高（在精英圈），而X不高（没上大学）， 那么他的Y必须很高（收入高）才能「解释」为什么他在精英圈。

这就造成了虚假的负相关：控制C后，X和Y出现负相关！

🚨 这是最反直觉的坏控制

对撞变量通常与X和Y都相关，看起来像是「好控制变量」。

但控制它会打开一条本来不存在的虚假因果路径。

经典例子：控制「当前职业」研究教育对收入的影响—— 因为职业被教育和收入共同决定，它是对撞变量！

🔴 第三类坏控制：M-偏倚（M-bias）

1 什么是M-偏倚？

M-偏倚是一种特殊的结构，控制变量Z看起来像是混淆变量， 但实际上是两个混淆变量的共同结果。

2 M-偏倚的结构

\( X \leftarrow U_1 \rightarrow \underbrace{Z}_{\text{控制变量}} \leftarrow U_2 \rightarrow Y \)

结构像字母「M」：两个混淆变量U1和U2都影响Z。

3 案例：地理位置作为M-偏倚变量

假设：

  • U1 = 家庭经济条件 → 影响大学选择（X）和居住地（Z）
  • U2 = 当地就业市场 → 影响居住地（Z）和收入（Y）

居住地Z与X和Y都相关，看起来应该控制...

但控制Z会打开路径：X←U1→Z←U2→Y，引入偏误！

🎲 M-偏倚的争议

M-偏倚在学界有争议。一些学者认为它在实际中很少见， 另一些学者认为它被低估了。

关键问题：U1和U2是否存在？如果存在，控制Z就是坏的； 如果不存在，Z可能是好的。

建议：如果不确定因果结构，对Z进行敏感性分析。

🎯 决策框架：如何判断控制变量的好坏？

第一步：绘制因果图（DAG）

在控制任何变量之前，先画出你假设的因果结构：

  1. 标出处理变量 X 和结果变量 Y
  2. 标出所有可能影响 X 或 Y 的变量
  3. 用箭头表示因果方向（基于理论和领域知识）
  4. 识别混淆路径（X←Z→Y）

第二步：应用后门准则（Back-door Criterion）

一个变量集合Z满足后门准则，当且仅当：

  1. Z阻断了所有从X到Y的「后门路径」（即以箭头指向X开始的路径）
  2. Z不阻断任何因果路径（即不以任何方式为X和Y的中介）
  3. Z不打开任何新的非因果路径（即不包含对撞变量）
变量类型 因果结构 是否控制 原因
混淆变量 X ← Z → Y ✅ 控制 阻断后门路径，消除偏误
结果预测变量 Z → Y，Z ⊥ X ✅ 控制 减少方差，提高精度
中介变量 X → Z → Y 🚫 不控制 阻断因果路径，低估总效应
对撞变量 X → Z ← Y 🚫 不控制 打开虚假路径，引入偏误
M-偏倚变量 X ← U1 → Z ← U2 → Y ⚠️ 通常不控制 可能打开M型路径

💡 实践建议

  1. 先画DAG，再跑回归：不要凭直觉选择控制变量
  2. 明确你的estimand：总效应还是直接效应？
  3. 后处理变量要谨慎：任何在处理之后测量的变量都可能是中介或对撞
  4. 报告敏感性：展示不同控制组合下的结果稳定性
  5. 使用工具：如 dagitty 软件检验后门准则

🔗 与张子尧、黄炜论文的连接

本文演示的是论文中的核心观点：

"控制变量的'好'与'坏'取决于其在因果结构中的位置...
控制变量的选择应由理论驱动而非数据驱动。"

关键 takeaway：

  • 不要仅仅因为变量与Y相关就控制它
  • 不要仅仅因为变量是「前定」的就认为控制它是安全的
  • 始终基于因果结构（DAG）来决定控制策略

📚 本章小结

  • 中介变量（X→Z→Y）：控制它会阻断因果机制，低估总效应
  • 对撞变量（X→Z←Y）：控制它会打开虚假路径，引入不可预测偏误
  • M-偏倚变量（X←U1→Z←U2→Y）：控制它可能引入偏误，结构依赖
  • 核心原则：控制变量的选择必须基于因果结构（DAG），而非统计相关性
  • 实用建议：先画因果图，再跑回归；明确estimand（总效应vs直接效应）
